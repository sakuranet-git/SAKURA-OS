# SAKURA OS リリースノート

## v1.2.0 (2026-03-16)

### バグ修正・機能追加

- 壁紙ボタン修正：pointer-events:none で非表示時のクリックブロック問題を根本解決
- ニュースソースを Yahoo! ニュース（rss2json API）に変更、NHK をフォールバックに
- ニュース12件表示保証（取得件数が不足の場合は繰り返し表示）
- 壁紙アップロード機能：画像ファイルを base64 変換して localStorage 永続保存（バージョンアップ後も維持）

## v1.1.0 (2026-03-16)

### ダッシュボードUI全面リニューアル

- 左パネル追加：大型デジタル時計（秒表示）・日付・令和/干支・📍大阪市
- リアルタイム天気ウィジェット（wttr.in API / 大阪市）：気温・体感・湿度・風速・降水・3日間予報
- 上部ニュースティッカー：NHK RSSライブ取得（allorigins プロキシ経由）
- デスクトップに為替レートカード（USD/JPY・EUR/JPY・CNY/JPY）
- デスクトップにニュースグリッド（6件表示）
- 壁紙変更機能：プリセット9種 + カスタムURL入力（localStorage保存）
- アプリアイコンを左パネルに移動
