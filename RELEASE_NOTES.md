# SAKURA OS リリースノート

## v1.5.9 (2026-03-16)

### パフォーマンス最適化

- クロック更新（毎秒）の DOM クエリをキャッシュ化：`getElementById` × 4 を setInterval 外に移動
- Firestore `onSnapshot` リスナーを `beforeunload` で確実に解除（メモリリーク修正）
- 重複していた `DOMContentLoaded` ハンドラを1つに統合

## v1.5.8 (2026-03-16)

### UI改善：アプリアイコン Mac スタイル化

- 左パネルのアプリ一覧をリスト → 3列グリッドに変更
- 各アプリに個別カラーのグラデーション丸角アイコン（52px・光沢レイヤー付き）
- アイコン下にラベルを配置するデスクトップアイコン型レイアウト

## v1.5.7 (2026-03-16)

### マスコット最適化

- PNG が透明背景済みのため Canvas の白背景除去ループを削除（`clearRect` + `drawImage` のみに簡略化）

## v1.5.6 (2026-03-16)

### マスコット最適化（Gemini更新対応）

- マスコットサイズを 200px → 300px に拡大
- 吹き出し位置調整（`bottom: 310px`、`left: 80px`）
- `_initPosition()` から `_loadPosition()` 呼び出しが削除されていたバグを修正（ドラッグ位置がリロードで消える問題）

## v1.5.5 (2026-03-16)

### マスコット最適化（Gemini更新対応）

- マスコットサイズを 110px → 200px に拡大（canvas インラインサイズを修正）
- 吹き出し位置を新サイズに合わせて調整（`bottom: 210px`、`left: 60px`）
- 吹き出しの show/hide アニメーション不整合を修正（`translateX(-50%)` を削除）

## v1.5.4 (2026-03-16)

### マスコット背景完全透明化

- `mix-blend-mode` から Canvas API 方式に変更：白・淡色ピクセルを直接 alpha=0 に変換
- アンチエイリアス境界もグラデーション処理で自然に透明化

## v1.5.3 (2026-03-16)

### マスコット修正

- ドラッグ機能の呼び出し漏れを修正（`_initDrag()` が未実行だった）
- 背景透明化：ダークモードの `mix-blend-mode: screen` を削除し `multiply` に統一（白背景を正しく除去）

## v1.5.2 (2026-03-16)

### マスコット再修正

- ON/OFFトグルを壁紙パネル内から左パネル（常時表示）に移動

## v1.5.0 (2026-03-16)
