# SAKURA OS リリースノート

## v1.3.4 (2026-03-16)

### バグ修正

- カレンダー・本日の作業予定が反映されない問題を根本修正：`gw_api.php` 経由の設定取得・匿名認証を廃止し、`ordersystem.html` と同一方式（Firebase設定ハードコード・認証不要）に変更

## v1.3.3 (2026-03-16)

### バグ修正

- カレンダー・本日の作業予定が反映されない問題を修正：`gw_api.php` の相対パスを `groupware/gw_api.php` に修正（`/system/gw_api.php` ではなく `/system/groupware/gw_api.php` が正しいパス）

## v1.3.2 (2026-03-16)

### バグ修正

- 壁紙アップロードをIndexedDBで保存する方式に変更（グループウェアと同方式・LocalStorageの5MB制限を完全回避）
- 旧LocalStorage保存データからIndexedDBへの自動移行対応

## v1.3.1 (2026-03-16)

### バグ修正

- カレンダー・本日の作業予定が最新データに反映されない問題を修正：Firebase Auth SDKを追加し匿名サインインでFirestoreのセキュリティルールを通過するように対応

## v1.3.0 (2026-03-16)

### バグ修正・改善

- パネル・カード透明度を大幅強化（壁紙がしっかり透けて見えるレベルに）
- ニュース9枚目：取得件数が9未満の場合もループ表示して空欄なし

## v1.2.0 (2026-03-16)

### バグ修正・機能追加

- 壁紙ボタン修正：pointer-events:none で非表示時のクリックブロック問題を根本解決
- ニュースソースを Yahoo! ニュース（rss2json API）に変更、NHK をフォールバックに
- ニュース12件表示保証（取得件数が不足の場合は繰り返し表示）
- 壁紙アップロード機能：画像ファイルを base64 変換して localStorage 永続保存（バージョンアップ後も維持）

## v1.1.0 (2026-03-16)

### ダッシュボードUI全面リニューアル

- 左パネル追加：大型デジタル時計（秒表示）・日付・令和/干支・📍大阪市
- リアルタイム天気ウィジェット（wttr.in API / 大阪市）：気温・体感・湿度・風速・降水・3日間予報
- 上部ニュースティッカー：NHK RSSライブ取得（allorigins プロキシ経由）
- デスクトップに為替レートカード（USD/JPY・EUR/JPY・CNY/JPY）
- デスクトップにニュースグリッド（6件表示）
- 壁紙変更機能：プリセット9種 + カスタムURL入力（localStorage保存）
- アプリアイコンを左パネルに移動

- 右下ドラッグによるウィンドウリサイズ
